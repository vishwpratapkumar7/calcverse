🚀 CalcVerse — Ultimate Calculator Suite

A modern, all-in-one calculator web application that combines standard, scientific, financial, currency, and unit conversion tools into a single powerful and responsive platform.

---

🔥 Features

🧮 Core Calculators

- Standard Calculator with history
- Scientific Calculator (trigonometry, logarithms, powers, factorial, etc.)

💰 Finance Suite

- Compound Interest Calculator
- Simple Interest Calculator
- EMI / Loan Calculator
- SIP / Mutual Fund Calculator
- FD / RD Calculator

💱 Currency Converter

- 180+ currencies
- Live exchange rates with fallback system
- Quick comparison with major currencies

📐 Unit Converters

- Temperature (Celsius, Fahrenheit, Kelvin, Rankine)
- Length, Mass, Speed
- Area, Volume
- Data Size, Time

📊 Business & Utility Tools

- Percentage Calculator
- Profit / Loss Calculator
- Discount Calculator
- GST / Tax Calculator
- Age Calculator
- Ratio & Proportion

---

🧠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
